# 01-basics/01-hello-world

This example demonstrates the fundamental concepts of the Lustre framework 
running inside a tauri webview container.

## Quickstart

### Run it

```bash
cd [this directory]
npm install
npm run tauri dev
```

### Build it

```
npm run tauri build
```
